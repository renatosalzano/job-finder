#### Feature

- [x] Easy to use
- [x] Compact Search Bar
- [x] Minimal Design
- [x] Full responsive

---

#### Usage
1. `git clone [url]`
2. `cd [repo-folder-name]`
3. `npm install --save-dev angular-cli@latest`
4. `npm install`
6. `ng serve`
